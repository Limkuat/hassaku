# Hassaku
Hassaku is... strange.

Hassaku is a simple automation tool with a *very* specific use-case for Kandid.


## Usage

From file:
```console
hassaku /path/to/file.opus
...
```

From YouTube or Twitch:
```console
hassaku "https://www.youtube.com/watch?v=5fdSjFIG5VE"
...
```

## Dependencies
 * `zsh`
 * `ffmpeg`
 * `youtube-dl` (if used)
